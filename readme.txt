This plugins add Mailjet as Action output from elementor forms.

== Description ==

Manage your Mailjet subscribers with elementor forms. 


Add your API Key and Secret on wp-config.php
define( 'MAILJET_API', '' );
define( 'MAILJET_SECRET', '' );

Select a elementor form, add an action Mailjet, type the list ID you want your subscribers to be part of.
